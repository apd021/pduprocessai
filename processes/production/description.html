Production

Clause: 12.4.2

Objective

Apply cybersecurity controls during production to ensure items are manufactured securely.

Inputs

  • Production control plan
  • Security procedures
  • Cryptographic materials

Outputs / Work Products

  • WP-12-04: Production records
  • WP-12-05: Key provisioning records
  • WP-12-06: End-of-line test results

Requirements

  • [RQ-12-05] Production control plan shall be applied
  • [RQ-12-06] Security procedures shall be followed
  • [RQ-12-07] Deviations shall be documented and addressed
  • [RQ-12-08] Production data shall be protected

Key Activities

  • Execute secure programming and flashing
  • Provision cryptographic keys and certificates
  • Enable security features (secure boot, debug lock)
  • Perform end-of-line security verification
  • Maintain production traceability
  • Handle and dispose of sensitive materials

Security Checks

  • Verify correct firmware version
  • Confirm key provisioning success
  • Validate secure boot activation
  • Test security functionality
  • Verify debug interface lockdown
