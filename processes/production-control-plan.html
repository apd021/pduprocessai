Production Control Plan

Clause: 12.4.1

Objective

Define controls for the production phase to ensure cybersecurity is maintained during manufacturing.

Inputs

  • Cybersecurity requirements
  • Release for post-development
  • Production specifications

Outputs / Work Products

  • WP-12-01: Production control plan
  • WP-12-02: Security procedures for production
  • WP-12-03: Key management procedures

Requirements

  • [RQ-12-01] Production controls shall be defined
  • [RQ-12-02] Secure handling of sensitive data shall be ensured
  • [RQ-12-03] Production tools shall be secured
  • [RQ-12-04] Traceability shall be maintained

Key Activities

  • Define security controls for production environment
  • Establish key provisioning and management procedures
  • Define secure flashing and programming procedures
  • Implement access controls for production systems
  • Define handling of security-sensitive materials
  • Establish production audit procedures

Production Security Topics

  • Cryptographic key injection
  • Secure boot enablement
  • Certificate provisioning
  • Serial number and ID management
  • End-of-line security testing
