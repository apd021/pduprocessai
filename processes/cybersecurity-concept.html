Cybersecurity Concept

Clause: 9.5

Objective

Define the cybersecurity concept that describes how cybersecurity goals will be achieved through security controls and mechanisms.

Inputs

  • Item definition
  • Cybersecurity goals
  • Preliminary architecture

Outputs / Work Products

  • WP-09-06: Cybersecurity concept
  • WP-09-07: Cybersecurity requirements
  • WP-09-08: Security controls specification

Requirements

  • [RQ-09-09] Cybersecurity concept shall be defined
  • [RQ-09-10] Concept shall address all cybersecurity goals
  • [RQ-09-11] Security controls shall be specified
  • [RQ-09-12] Requirements shall be derived from the concept

Key Activities

  • Define security architecture principles
  • Specify security mechanisms and controls
  • Derive cybersecurity requirements
  • Allocate requirements to components
  • Define security verification requirements
  • Review concept for completeness

Typical Security Controls

  • Authentication and authorization
  • Cryptographic protection
  • Secure communication protocols
  • Secure boot and firmware verification
  • Intrusion detection and logging
  • Access control and segmentation
