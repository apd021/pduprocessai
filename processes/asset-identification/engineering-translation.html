Engineering Translation: Asset Identification

Technical Implementation

Systematically identify and catalog cybersecurity-relevant assets within the item using structured analysis techniques.

Implementation Approach

  • Architecture Analysis: Review system architecture diagrams, data flow diagrams, and interface specifications
  • Data Classification: Identify sensitive data (keys, credentials, personal data) using data flow analysis
  • Function Analysis: Catalog security-critical functions (authentication, authorization, secure boot)
  • Component Mapping: Map assets to physical components (ECUs, sensors, communication modules)
  • Threat Modeling Tools: Use tools like Microsoft Threat Modeling Tool or OWASP Threat Dragon

Technical Specifications

  • Asset catalog template with fields: ID, name, type, location, CIA properties
  • Data flow diagrams (DFD) showing data movement
  • System architecture diagrams with asset annotations
  • Asset-to-component mapping matrix
  • Damage scenario templates

Engineering Artifacts

  • WP-15-01: Asset identification report
  • WP-15-02: Asset catalog (spreadsheet or database)
  • WP-15-03: Damage scenarios for each asset
  • Architecture diagrams with asset annotations
  • Data flow diagrams

Tools and Methods

  • Threat modeling tools (Microsoft TMT, OWASP Threat Dragon)
  • Architecture modeling tools (Enterprise Architect, MagicDraw)
  • Data flow analysis tools
  • Asset management databases
  • Diagramming tools (Draw.io, Visio)

Asset Categories

  • Data Assets: Cryptographic keys, certificates, personal data, configuration data
  • Function Assets: Authentication, secure boot, intrusion detection, access control
  • Component Assets: ECUs, gateways, sensors, actuators, communication interfaces
  • Communication Assets: CAN bus, Ethernet, V2X, cellular, WiFi, Bluetooth

Cybersecurity Properties

  • Confidentiality (C): Protection from unauthorized disclosure
  • Integrity (I): Protection from unauthorized modification
  • Availability (A): Protection from denial of service
