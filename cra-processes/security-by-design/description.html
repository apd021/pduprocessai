Security by Design

Article: 10(1)

Objective

Design and develop products with cybersecurity built in from the start, ensuring security is an integral part of the product lifecycle.

Inputs

  • Product requirements
  • Threat model
  • Security requirements

Outputs / Work Products

  • Security architecture
  • Secure design documentation
  • Security requirements specification

Requirements

  • Products shall be designed with security as a fundamental consideration
  • Security requirements shall be defined early in the development process
  • Security controls shall be integrated into the product design

Key Activities

  • Define security requirements during design phase
  • Apply secure design principles
  • Integrate security controls into architecture
  • Conduct security design reviews
